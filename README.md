# Lineup

Lineup is a timeline UI to plan tasks per day (video + notes), with infinite scrolling around Today.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

### Environment

Create `.env.local` (you can copy `.env.example`) and set:

- `MONGODB_URI`
- `MONGODB_DB` (optional, defaults to `lineup`)
- `YOUTUBE_API_KEY` (optional; required in **production** for video duration â€“ YouTube blocks server requests from ytdl-core. Get a key from [Google Cloud Console](https://console.cloud.google.com/apis/credentials) with YouTube Data API v3 enabled.)

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
